Premier Stage







L’entreprise dans laquelle j’ai effectué mon stage s’appelle GS Optic il s’agit d’un opticien qui possède plusieurs boutique : Paris Belleville, Montreuil, Paris Maubert. Cet opticien propose divers types de lunettes de vue ou de soleil, des lentilles mais aussi des paires de lunettes personnalisables. Le chef d’entreprise s’appelle Gregory SAYADA, chacune de ses boutiques est accompagné d’au moins deux opticiens spécialisés. J’ai eu pour projet de crée un nouveau site pour l’entreprise



Pour la création du site l’entreprise à souhaité que j’utilise le logiciel WordPress. Le site doit comprendre une page de présentation de l’entreprise constitué d’un menu avec plusieurs pages, une page pour contacter l’opticien, une page récapitulative des différentes boutiques et différentes pages qui présentes les différents produit vendu par l’entreprise. Le site doit être fonctionnel sur tous types d’appareil du smartphone jusqu’à l’ordinateur. Le site doit respecter les couleurs de l’entreprise ( noir, rouge et blanc ) Ce dernier doit être épuré et facile d’utilisation.






Voici le résultat du site : GS OPTIC



  • Conclusion

    Ce stage m’a beaucoup appris sur ce qu’était la création d’un site web complet pour une entreprise. J’ai par ailleurs appris à utiliser divers logiciel tel que WordPress, Elementor ou encore Photoshop. Ce stage m'a beaucoup enrichie sur les responsabilités que l’on pouvait avoir dans un projet au sein d’une entreprise.
