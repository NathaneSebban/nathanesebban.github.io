Premier Stage







L’entreprise dans laquelle j’ai effectué mon stage s’appelle GS Optic, il s’agit d’un opticien qui possède plusieurs boutiques : Paris rue Belleville, Montreuil, Paris rue Maubert. Cet opticien propose divers types de lunettes de vue ou de soleil, des lentilles mais aussi des paires de lunettes personnalisables. Le chef d’entreprise s’appelle Gregory SAYADA, chacune de ses boutiques est accompagné d’au moins deux opticiens spécialisés. J’ai eu pour projet de créer un nouveau site pour l’entreprise



Pour la création du site, l’entreprise a souhaité que j’utilise le logiciel WordPress. Le site doit comprendre une page de présentation de l’entreprise constitué d’un menu avec plusieurs pages. Une page pour contacter l’opticien, une page récapitulative des différentes boutiques et différentes pages qui présentes les différents produits vendus par l’entreprise. Le site doit être fonctionnel sur tous types d’appareils du smartphone jusqu’à l’ordinateur. Le site doit respecter les couleurs de l’entreprise (noir, rouge et blanc) Ce dernier doit être épuré et facile d’utilisation.






Voici le résultat du site : GS OPTIC



  • Conclusion

    Ce stage m’a beaucoup appris sur ce qu’était la création d’un site web complet pour une entreprise. Ce stage m'a beaucoup enrichi sur les responsabilitées que l’on pouvait avoir dans un projet au sein d’une entreprise.
